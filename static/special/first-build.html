.env

Мы создали для Вас этот файл, однако его содержимое вы должны заполнить сами.

Ведь в нём хранятся секретные токены и другая дребедень.

Ищите в корневой папке: ./.env

Возможно, вы захотите настроить конфиг,
./config.json.js

Самые важные параметры:

    server.origin — необходим для корректной работы сервера.

Существующие проблемы

Этот проект не задумывался как воспроизводимый. Поэтому запуск с другими данными череват некоторыми проблемами

    1.

    Ваш бот не знает о существовании Наших эмодзи.

    Это черевато частыми ошибками и текстовыми идентификаторами в сообщениях вместо самих картинок.

    Способ исправления:

    Замените все идентификаторы в коде на собственные:

    1. В редакторе кода Комбинацией CTRL + SHIFT + F откройте глобальный поиск (работает в большинстве редакторов)

    2. Во-первых, замените совпадения <a:name:id>, например, <:arrowleft:640449848050712587> на аналогичную конструкцию, где a — анимирован ли эмодзи, name — его имя, и id — его идентификатор

    3. Во-вторых, сделайте тоже самое со всеми идентификаторами, то есть замените, например 640449848050712587, на айди вашего эмодзи

Успех?

Используйте npm run start и начните исследовать!
